먼저 흐름은

0. example.server.com같이 내가 만든 주소로 들어온다
1. 로비월드로 이동된다
2. 로비월드에서 npc나 포탈등의 상호작용을 할 수 있다.
3. 로비월드는 파괴와 설치가 불가능하며 공간이 제한된다.
4. 로비월드는 나 또는 내가 임명한 사람만 수정할수있다.
5. 로비월드에서 npc와의 상호작용으로 월드리스트가 뜨며 페이징기능이 존재한다.
6. 탭으로 내 월드를 눌러서 내 월드(최대3개까지 생성가능)에 들어갈수있다.
7. 최초 사용자는 월드가 업으므로 월드생성하기로 월드를 만든다.
8. 월드를 생성할때 비밀번호 여부, 비밀번호 체크하면 비밀번호 설정, 월드명, 월드타입(최초 설정후 수정불가능), 화이트리스트 여부, 최대 플레이어수 등등 미리 설정하게한다.
9. 사용자마다 인스턴트 서버를 하나씩 생성하고 서버하나에 3개의 월드까지 사용가능하며 접속자가 없을경우 서버는 꺼진다.
10. 내 월드에서 특정 행동으로 ui가 나타나서 월드명수정이나 비밀번호수정, 비밀번호 끄기 등등으로 기본설정을 바꿀수있다. (월드타입은 불가능)
11. 친구목록이 존재하며 친구월드 이동하기 기능이 있다.
12. 월드리스트에는 비밀번호가 있는방이면 표시가 되어있으며 비밀번호를 통과해도 화이트리스트까지 있는방이면 거기서 제한이 걸린다.
13. 플러그인 추가 ui가 있어서 원하는 플러그인을 체크하면 해당 서버는 재실행되면서 플러그인이 추가된다.
14. 일렉트론앱은 나만이 보는 전체적인 흐름을 보는곳이며 웹소켓을 통해서 마인크래프트와 통신할 수 있다.

```
[ Lobby Plugin ]        [ World Server Plugins ]
        │                         │
        └──── Plugin Message ────┘
                     │
          [ Central Control Plugin ]
                     │
        ┌────────────┴────────────┐
        │                         │
   [ Node.js Control Plane ]   [ Electron ]
```
