# Schedule

## #1 2025-09-15

### 1) 우선순위 작업 흐름

1. 로비 월드 리스트/페이징 UI 구현
2. 월드 생성 기본 플로우(이름/타입/비밀번호/최대인원)
3. Core 상태 저장(메모리) + DB 스키마 초안 작성
4. 월드 서버 인스턴스 생성/종료 트리거(더미 구현)

### 2) 검증 포인트

- 로비에서 월드 리스트가 정상 표시되고 페이지 전환이 된다.
- 월드 생성 요청이 Core 상태에 반영되고, 기본 설정이 저장된다.
- Core가 UUID/월드 정보를 조회 가능하다.
- 월드 서버 기동/종료 흐름이 호출 가능하다(실제 기동은 추후).
