# Mcger Plugins Progress Log

- Added a world list UI in the lobby plugin (inventory-based, paged).

- Cleaned up Eclipse metadata and stabilized jdtls root behavior for this repo.

## Runtime Notes

- The world list UI reads from `RuntimeContext.getWorldSummaries()`.
- JSON data is loaded from the plugin data folder:
  - `~/minecraft/servers/lobby/plugins/McgerCorePlugin/worlds.json`
- The resource `worlds.json` is copied into the data folder on first run.

## Environment / Tooling

- Removed Eclipse metadata from repo root to avoid classpath conflicts:
  - `/.classpath`, `/.project`, `/.settings/`
- `package.json` script `lsp` runs `./gradlew eclipse` and will recreate those files.
