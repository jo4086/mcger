# Mcger 플러그인 작업 로그

## 오늘 한 일 요약

- 네비게이터 UI 뼈대를 구성하고, 로비 플러그인에서 UI 책임을 분리함.
- 월드 목록 UI를 네비게이터 플러그인으로 옮길 준비를 진행함.
- Core에서 JSON 기반 월드 목록 로딩을 추가해 대량 테스트 데이터 표시 가능하게 함.
- LSP/jdtls 문제를 해결하기 위해 Eclipse 설정 파일 정리 및 루트 설정을 점검함.

## 코드 변경 내용

### 1) Navigator UI 뼈대

- 메인 메뉴 UI(Servers / My World / Friends / Lobby) 구조 추가
- 메뉴 클릭 이벤트 처리 및 임시 화면(Placeholder) 구성

주요 파일(네비게이터 쪽):
- `navigator-plugin/src/main/java/io/mcger/navigator/ui/MainMenu.java`
- `navigator-plugin/src/main/java/io/mcger/navigator/ui/MainMenuHolder.java`
- `navigator-plugin/src/main/java/io/mcger/navigator/ui/PlaceholderMenu.java`
- `navigator-plugin/src/main/java/io/mcger/navigator/listener/MainMenuListener.java`
- `navigator-plugin/src/main/java/io/mcger/navigator/command/WorldsCommand.java`
- `navigator-plugin/src/main/java/io/mcger/navigator/McgerNavigatorPlugin.java`

### 2) Lobby 플러그인 정리

- 로비 플러그인에서 UI/커맨드/리스너 관련 코드 제거
- 로비 플러그인은 로비 보호/조인 처리로만 유지

수정 파일:
- `lobby-plugin/src/main/java/io/mcger/lobby/McgerLobbyPlugin.java`

### 3) Core JSON 월드 데이터 로딩

- `worlds.json` 파일을 통해 임시 월드 목록을 로딩
- 서버 실행 시 `plugins/McgerCorePlugin/worlds.json` 경로에서 읽음

관련 파일:
- `core-plugin/src/main/java/io/mcger/core/McgerCorePlugin.java`
- `core-plugin/src/main/resources/worlds.json`

## 실행/테스트 메모

- `/worlds` 명령은 네비게이터 메인 UI로 진입
- `Servers` 버튼은 월드 리스트 UI로 연결 예정
- 테스트 데이터는 Core의 JSON에서 로딩됨

## 정리/환경

- Eclipse 관련 파일 제거: `/.classpath`, `/.project`, `/.settings/`
- `npm run lsp`는 `./gradlew eclipse`로 위 파일을 재생성함
